Project Overview

Wingy is a feature-rich online multiplayer game designed for both competitive and casual play, where players can challenge each other in real-time using fun, fast-paced mechanics.

Key Features:

  • Game Modes: Includes both a casual "Classic" mode and a competitive "Ranked" mode.
  • Reward System: Daily rewards encourage return players and engagement.
  • Leaderboards: Weekly and monthly rankings promote friendly competition.
  • In-Game Shop: Players can purchase diamonds, tickets, gold, and new characters.
  • Customization: Unlock and select from a variety of unique characters.

Technologies Used:

Unity C# Photon PlayFab
