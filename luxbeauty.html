Detail Proyek

LuxBeauty adalah situs web e-commerce yang dibangun menggunakan framework CodeIgniter 4. Proyek ini dikembangkan sebagai tugas akhir untuk mata kuliah Pemrograman Web Interaktif 2.

Fitur Utama:

  • Fungsionalitas E-commerce Lengkap: Autentikasi pengguna, katalog produk, keranjang belanja, dan proses checkout.
  • Dasbor Admin: Panel backend untuk mengelola produk, pesanan, dan pengguna.
  • Arsitektur MVC: Dibangun mengikuti pola Model-View-Controller dengan CodeIgniter 4.
