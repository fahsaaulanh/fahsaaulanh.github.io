Project Details

LuxBeauty is an e-commerce website built with the Codeigniter 4 framework. This project was developed as a final assignment for the Interactive Web Programming 2 course.

Key Features:

  • Full E-commerce Functionality: User authentication, product catalog, shopping cart, and checkout process.
  • Admin Dashboard: A backend panel for managing products, orders, and users.
  • MVC Architecture: Built following the Model-View-Controller pattern with Codeigniter 4.
