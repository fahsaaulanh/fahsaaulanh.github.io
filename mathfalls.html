Project Overview

MathFalls is a 3D platformer game infused with fun and challenging mathematics-based gameplay. The game features multiple levels, each offering unique mechanics and increasing levels of difficulty. Players are required to solve math-related tasks while navigating through engaging platforming environments.

Designed to combine entertainment and education, MathFalls encourages logical thinking and quick reflexes, making it suitable for both casual players and learners. As players progress, they will encounter new challenges that test their problem-solving abilities in a dynamic and interactive way.

Key Features:

  • Online Leaderboard: Displays the top 100 high scores and player names globally.
  • User Authentication: Secure login and data storage powered by PlayFab services.
  • Character Selection: Players can choose from multiple playable characters.
  • Versatile Controls: Supports both keyboard and joystick input for flexible gameplay.
  • Progressive Levels: Each stage introduces new math mechanics and increasing challenges.
